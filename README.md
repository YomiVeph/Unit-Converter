Unit Converter App
Overview

The Unit Converter App is a sleek, responsive web application that allows users to convert between different units of measurement easily. It supports conversions for Length, Weight, Volume, and Temperature, and provides a modern, interactive interface with live conversion and a swap feature.

This app is ideal for students, professionals, and hobbyists who need quick and accurate unit conversions.

Features

Multiple Unit Types: Convert between Length, Weight, Volume, and Temperature units.

Live Conversion: Results update in real-time as users type the value or change units.

Swap Functionality: Swap "From" and "To" units instantly with a single click.

Responsive Design: Fully functional and visually appealing on desktop, tablet, and mobile devices.

Dark Theme UI: Modern dark design with smooth hover and focus states.

Extensible: Easy to add new units or categories in the future.

Supported Units

Length: meter, kilometer, centimeter, millimeter, mile, yard, foot, inch
Weight: kilogram, gram, milligram, pound, ounce
Volume: liter, milliliter, gallon, cup
Temperature: Celsius, Fahrenheit, Kelvin

Open index.html in your preferred browser.

How It Works

The app stores conversion factors for each unit type in JavaScript objects.

When a user enters a value or selects units, the app calculates the conversion based on a base unit.

For temperature, the app uses the correct formula conversions (Celsius ‚Üî Fahrenheit ‚Üî Kelvin).

Results are displayed instantly, updating live without page reloads.
